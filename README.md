# Stackline Product Page

## Technologies Used

### 1. **Next.js (15.1.6)**

### 2. **React (19.0.0) & React DOM (19.0.0)**

### 3. **TypeScript**

### 4. **Tailwind CSS & UI**

### 5. **React Chart.js 2 (5.3.0)**

### 6. **Day.js (1.11.13)**

## Installation

To install the dependencies, run the following command:

```bash
npm install
```

or using Yarn:

```bash
yarn install
```

## Usage

### Running the Development Server

To start the development server:

```bash
npm run dev
```

or

```bash
yarn dev
```
